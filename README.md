# EDC-Preparation
用于上传整理好的代码、学习内容以及相关资料

注意：
- 仅上传代码到对应项目文件夹,资料类的东西放在其它的文件夹内
- 请检查后在 push ，正在测试或未完成的代码重新拉一个分支，最后确认正确后再 merge 到 main

## 文件路径
```
- control       //控制
  |- Tb6612
    |- driver
    |- ...
  |- ...        //不同传感器分开放
- vision        //视觉
- hardware      //硬件
- reference     //其它资料
- trying        //其它尝试的小项目
```

## 控制代码
- 注意封装，面向对象模块化编程，封装好的函数适当添加注释
- 尽量做引脚映射，防止更换引脚时混乱
- readme 注明传感器，不同传感器分开
